## Getting Started

Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying app/page.tsx.

## Notes

- This project uses the **Next.js App Router** (`app/` directory).
- It is a **static landing page** and does not include backend APIs or database integration.
- All UI logic lives in React components under `components/`.
